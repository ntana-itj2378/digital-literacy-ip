# 知的財産権リスク啓蒙サイト プロジェクト引き継ぎ報告書 (v2.0)

作成日：2026年4月23日
作成者：エンジニア（Antigravity）

本資料は、2026年4月20日版の報告書以降に実施された大規模なUI刷新、機能拡張、およびコンテンツの構造化を反映した最新の引き継ぎ資料です。

---

## 1. 最近の主要なアップデート (2026/04/21 - 04/23)

直近の数日間で、サイトの信頼性と操作性を飛躍的に高める以下のアップデートを実施しました。

### 1-1. UI/UXの「警戒モード」刷新
- **ホーム画面**: 従来のリスト表示から、3つの主要カテゴリ（トレント、ファスト動画、海賊版）を巨大なパネルで選択する直感的なグリッドUIへ変更。
- **警戒デザイン**: プレミアム・ダークテーマをベースに、警告ストライプやゴールドのグロー効果を用いた「警戒UI」を導入し、法的リスクを視覚的に強調。
- **カードデザイン**: 証拠資料を模した「FILE XX」形式のカードデザインを採用。タグ表示を整理し、ノイズを排除したクリーンな表記へ統一。

### 1-2. ナビゲーションの高度化
- **History APIの導入**: URLハッシュと連動した表示切り替えを実装。SPA構成でありながらブラウザの「戻る／進む」ボタンで直感的に遷移可能。
- **記事間ナビゲーション**: 各記事末尾の「次の記事へ」ボタンを自動化。ルート相対パスを用いることで、本番環境でも確実に次のコンテンツへ直接遷移する動線を確立。
- **ホワイトテキスト化**: 記事本文およびサブタイトルの視認性を最大化するため、主要テキストを白字（#fff）へ統一。

### 1-3. 知財Wikiの専門データベース化
- **3軸構造化**: Wikiコンテンツを「法律」「ツール」「サイト・事件」の3セクションに明確に分割。
- **詳細解説の追加**: 著作権法119条（刑事罰）、民法709条（損害賠償）、BitTorrentの技術的リスク、VPNの匿名性に関する誤解など、専門性の高い解説を大幅に拡充。

## 2. 実装済みコンテンツと構造

### 2-1. カテゴリ別解説セクション
各カテゴリの記事一覧の冒頭に、読者が事例を読む前に基本知識を得るための解説を導入しました。
- **トレント**: P2Pの自動アップロードによる「送信可能化権」侵害のリスク。
- **ファスト動画**: 作品価値を毀損する翻案権侵害と、億単位の巨額賠償事例。
- **海賊版**: 令和3年法改正による静止画ダウンロード刑事罰化とセキュリティリスク。

### 2-2. 記事の整理
- 全14記事から「締めの言葉」といった冗長な小見出しを削除し、本文が独立して流れる洗練された構成に変更。
- 外部リンク（警察庁、CODA、法律事務所等）をオレンジ色（#ff9900）に統一し、外部リソースへの導線を明確化。

## 3. 技術的仕様と保守

### 3-1. ビルド環境
- **エンジン**: Node.js + Nunjucks (`build.js`)
- **開発ツール**: Viteを利用したホットリロード環境。環境変数注入（`Environment Injection`）により開発・本番の差異を吸収。
- **デプロイ**: VercelへのCI/CDパイプライン。`npx vercel --prod` で即座に最新版を公開。

### 3-2. ディレクトリ構成
- `src/`: ソースファイル（テンプレート、CSS、JS）
- `src/articles/`: 各記事（カテゴリ別にフォルダ分け）
- `src/wiki/`: Wikiポータル
- `dist/`: ビルド済み成果物（自動生成）
