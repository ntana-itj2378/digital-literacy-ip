FILE 03-03: 映画の「文字抜き出し」ネタバレサイトに有罪判決

基本ステータス

  • FILE ID: FILE 17
  • カテゴリ: ネタバレ・翻案権侵害
  • 重要度: ⚠️ 重大（最新の有罪判決事例）
  • タグ: #ネタバレサイト, #文字抜き出し, #有罪判決, #翻案権

タイムライン：事件の経緯

  1. サイト運営: 被告（39歳・会社経営）は、映画やアニメのあらすじ、台詞、場面展開を克明に文章でまとめ、関連画像とともに自身のウェブサイトに投稿。
  2. 収益化: これら「ネタバレ記事」により多額の広告収入を得ていた。
  3. 有罪判決: 2026年4月16日、東京地方裁判所は「映画の商品価値を失わせ、制作者の収益構造を破壊しかねない」として、著作権法違反（翻案権・複製権侵害）の罪で有罪判決を言い渡した。

法的リスクの分析

この判決は、「動画（ファスト映画）」だけでなく「文章」による詳細なネタバレも犯罪となることを明確に示しました。

A. 「引用」と「侵害」の境界線

  • ストーリーの全容公開: 作品を観なくても内容がすべて分かるほど詳細に書く行為は、法的な「引用」の範囲を大きく逸脱します。
  • 翻案権の侵害: 元の著作物の本質的な特徴を残したまま文章化する行為は「翻案（ほんあん）」にあたり、著作者の許諾が必要です。

B. 収益化の悪質性

  • 広告収入目的: 権利者に無断でコンテンツの価値を利用し、営利目的で公開することは、刑事罰の対象となる可能性が極めて高いです。

警戒すべきポイント

「文章なら大丈夫」「感想の範囲内」という自己判断は危険です。

[!CAUTION] 映画やアニメのストーリー、台詞の展開を丸ごと文章にする行為は、もはや「紹介」ではなく「侵害」です。特に収益化が絡む場合、厳格な法的措置が取られるフェーズに入っています。
