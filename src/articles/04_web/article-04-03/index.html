FILE 04-03: ディズニー・ユニバーサルがAI「Midjourney」を提訴

基本ステータス

  • FILE ID: FILE 15
  • カテゴリ: 生成AI・法的論争
  • 重要度: ⚠️ 重大（ハリウッドメジャーによる国際訴訟）
  • タグ: #Disney, #Midjourney, #生成AI, #著作権侵害

タイムライン：事件の経緯

  1. 提訴の発生: 2025年6月11日（現地時間）、エンターテインメント大手のウォルト・ディズニーとユニバーサル・スタジオが、画像生成AI大手「Midjourney（ミッドジャーニー）」を著作権侵害で提訴したことが報じられた。
  2. 主な主張: ハリウッドのメジャースタジオが保有する膨大な映画、キャラクター、美術設定が、権利者の許諾なくAIの学習データとして使用されたと主張。
  3. 具体的な侵害: AIによって生成された画像が、既存の有名キャラクター（ミッキーマウス等）と酷似しており、著作者の排他的な権利を侵害しているとしている。

法的リスクの分析

この訴訟は、生成AIの技術そのものではなく「権利物の取り扱い」を正面から問うものです。

A. AI開発・運営側の責任

  • 無断学習の是非: 著作権物を学習データとして利用する際、権利者の許諾が必要かどうかが最大の争点。
  • 翻案権・複製権侵害: オリジナルに酷似した画像を生成・出力させる仕組みを提供することの法的責任。

B. 利用者が直面するリスク

  • 類似性の問題: 「AIが作ったから自分のオリジナルだ」と思い込み、既存キャラクターに酷似した画像を商用利用・配布した場合、利用者自身が著作権侵害の当事者となる。
  • 国際的な権利行使: ディズニーのような世界的企業は、国境を越えて厳格な権利行使を行うため、訴訟に巻き込まれるリスクは極めて高い。

警戒すべきポイント

AIツールは便利ですが、その出力結果が「誰かの権利を侵害していないか」を判断する責任は、最終的に利用者にあります。

[!CAUTION] 生成AIで出力された画像が、偶然であれ意図的であれ有名キャラクターに似てしまった場合、それを公開・販売する行為は法的に極めて高いリスクを伴います。
